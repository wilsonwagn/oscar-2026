# 🏆 Aposta Oscar 2026

> Site interativo de apostas para a cerimônia do Oscar 2026 — Wilson & Vinicius

## ✨ Funcionalidades

- 🎬 **Página de Categorias** — escolha até 3 filmes por categoria (🥇 🥈 🥉)
- 📋 **Regras & Prêmios** — sistema de pontos, prêmios e consequências
- 💾 **Salva automaticamente** — escolhas guardadas no navegador (localStorage)
- 🖨️ **Exportar PDF** — gere um PDF com todas as suas apostas

## 📁 Estrutura

```
oscar-2026/
├── index.html       # Página inicial
├── page_1.html      # Regras & Prêmios
├── page_2.html      # Categorias (interativo)
├── css/
│   ├── shared.css   # Estilos compartilhados
│   ├── index.css    # Landing page
│   ├── page1.css    # Regras
│   └── page2.css    # Categorias + Print
└── js/
    └── page2.js     # Lógica de seleção e PDF
```

## 🎯 Como usar

1. Acesse o site
2. Digite seu nome na barra do apostador
3. Em cada categoria, clique nos filmes para definir sua 1ª, 2ª e 3ª opção
4. Clique em **"🖨️ Gerar PDF"** para salvar suas apostas
5. Compare os PDFs na noite da cerimônia!
